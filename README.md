The production-grade frontend for Indian Trade Market B2B platform. Enhanced features, CI/CD pipeline, and optimized deployment.

## ✨ Features

- **Complete B2B Marketplace** — Full trading platform UI
- **Enhanced Features** — Improved over MVP version
- **CI/CD Pipeline** — GitHub Actions automated deployment
- **Jest Testing** — Comprehensive test suite
- **API Integration** — Fixed auth endpoints and CSP headers

## 🚀 Quick Start

```bash
git clone https://github.com/dipk2003/final--fronted-itm.git
cd final--fronted-itm
npm install
npm run dev
```
